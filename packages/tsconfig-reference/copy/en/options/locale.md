The locale used when displaying messages to the user (e.g. 'en-us')


### Locales supported as of TypeScript 3.7

| Name                    | Locale  |
|-------------------------|---------|
| Brazilian Portuguese    | `pr-BR` |
| Chinese (Simplified)    | `zh-TW` |
| Chinese (Traditional)   | `zh-CN` |
| Czech                   | `cs-CZ` |
| English (US)            | `en-US` |
| French                  | `fr-FR` |
| German                  | `de-DE` |
| Italian                 | `it-IT` |
| Japanese                | `ja-JP` |
| Korean                  | `ko-KR` |
| Polish                  | `pl-PL` |
| Russian                 | `ru-RU` |
| Spanish                 | `es-ES` |
| Turkish                 | `tr-TR` |
